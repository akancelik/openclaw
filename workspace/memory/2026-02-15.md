### YouTube über OpenClaw - IPv6 Konfiguration
**Problem:** YouTube blockiert IPv4 für yt-dlp (Bot-Erkennung)

**Lösung:**
1. yt-dlp Config (`/docker/openclaw-rmys/data/.config/yt-dlp/config`):
   - `--cookies /data/.config/yt-dlp/cookies.txt`
   - `--extractor-args youtubepot-bgutilhttp:base_url=http://bgutil-provider:4416`
   - `--force-ipv6`

2. Docker IPv6 aktivieren (`/etc/docker/daemon.json`):
   ```json
   {
     "ip6tables": true,
     "experimental": true,
     "fixed-cidr-v6": "fd00::/80"
   }
   ```

3. Docker Neustart: `sudo systemctl restart docker`

**Ergebnis:** YouTube blockiert IPv6 NICHT!
